# 投資銘柄の選定・提案スキル

> **免責事項**: ここで提供する情報はすべて教育・分析目的であり、投資助言ではありません。
> 過去のパフォーマンスは将来のリターンを保証しません。最終的な投資判断はユーザ自身の責任で行ってください。

## トリガー

以下のようなユーザの発言でこのスキルを起動する:

- 「おすすめの銘柄を教えて」「買うべき株は？」
- 「直近で投資した方がいい銘柄」
- 「有望な銘柄を選定して」「注目銘柄は？」
- 「ポートフォリオを提案して」
- 「市場環境を分析して銘柄を提案して」

## 基本ルール

| ルール | 内容 |
|--------|------|
| リスク管理最優先 | 利益よりも損失回避を重視 |
| 断定禁止 | 「確実に儲かる」等の表現は絶対に使わない |
| 不確実性の明示 | 常に「〜が有望と考えられる」「〜のリスクがある」等の表現を使う |
| 免責事項 | 回答の冒頭に必ず免責事項を記載する |
| データソース明示 | 情報源の信頼性と制約を必ず明記する |

## ツール

| ツール | 用途 |
|--------|------|
| **Web検索** | マクロ環境、指数動向、金利・為替、地政学リスクの最新情報を取得 |
| **Python (yfinance等)** | 株価データの取得、リスク指標の定量算出（オプション） |

## 実行ステップ

| # | ステップ | 内容 | 検索クエリ例 |
|---|---------|------|-------------|
| 1 | **マクロ環境の確認** | Web検索で直近の市場データを収集 | `日経平均 S&P500 TOPIX NASDAQ 直近 動向` / `日銀政策金利 米国FF金利 USD/JPY 為替` |
| 2 | **マクロサマリー作成** | 主要指数・金利・為替・地政学リスクを表形式でまとめる | — |
| 3 | **直近の注意イベント収集** | 今後1〜2ヶ月の重要イベントをWeb検索で収集し、投資判断への影響度を評価する | `株式市場 注目イベント 経済指標 決算発表 今週 来週` / `日銀会合 FOMC 日程 スケジュール` |
| 4 | **有望セクター特定** | マクロ環境とイベントカレンダーを踏まえ追い風セクターを3〜5つ選定 | `注目セクター 有望銘柄 日本株 米国株` |
| 5 | **銘柄スクリーニング** | セクターからスクリーニング条件を満たす銘柄を日米各3〜5銘柄選定 | — |
| 6 | **個別銘柄詳細分析** | 基本情報、リスク指標、ファンダメンタルズ、カタリスト、リスク要因を分析 | — |
| 7 | **モデルポートフォリオ提案** | コア・サテライト戦略で配分比率を提案 | — |
| 8 | **総合コメント** | 200〜300字で投資判断のポイントをまとめる | — |

## Step 1: マクロ環境の確認

Web検索で以下を収集する:

- **主要指数**: 日経平均、TOPIX、S&P500、NASDAQ の直近値とトレンド
- **金利動向**: 日銀政策金利、米国FFレート、今後の利上げ/利下げ見通し
- **為替動向**: USD/JPY の水準と方向性
- **注目イベント**: 決算発表、中央銀行会合、地政学リスク、規制動向

## Step 2: マクロサマリー作成

以下の表形式で出力する:

```
| 指標 | 現在値 | トレンド |
|------|--------|----------|
| 日経平均 | XXXXX円 | （状況を記述） |
| TOPIX | XXXX | （状況を記述） |
| S&P 500 | X,XXX | （状況を記述） |
| 日銀政策金利 | X.XX% | （見通しを記述） |
| 米FF金利 | X.XX% | （見通しを記述） |
| USD/JPY | XXX円 | （方向性を記述） |
```

## Step 3: 直近の注意イベント収集

Web検索で今後1〜2ヶ月の重要イベントを収集し、以下の表形式で出力する。

### 検索クエリ例

- `株式市場 注目イベント 経済指標カレンダー 今週 来週`
- `日銀金融政策決定会合 FOMC 日程 スケジュール 2026`
- `決算発表 主要企業 スケジュール`
- `地政学リスク 注目ニュース 最新`

### 収集対象のイベントカテゴリ

| カテゴリ | 例 |
|----------|----|
| **金融政策** | 日銀会合、FOMC、ECB理事会、政策金利発表 |
| **経済指標** | 雇用統計、CPI、GDP、ISM景況指数、日銀短観 |
| **企業決算** | 主要企業の決算発表（特にマグニフィセント・セブン、日本の時価総額上位企業） |
| **政治・規制** | 関税政策の動向、選挙、ガバナンス・コード改訂、税制改正 |
| **地政学リスク** | 軍事衝突リスク、貿易摩擦、制裁、資源供給リスク |
| **市場イベント** | SQ（特別清算指数）、指数リバランス、IPO |

### 出力フォーマット

```
| 日付 | イベント | カテゴリ | 影響度 | 想定される影響 |
|------|---------|----------|--------|---------------|
| MM/DD | イベント名 | カテゴリ | 高/中/低 | 株式市場・為替への想定影響を簡潔に記述 |
```

### 影響度の判定基準

| 影響度 | 基準 |
|--------|------|
| **高** | 市場全体の方向性を変え得るイベント（金融政策変更、重要指標のサプライズ、地政学的有事） |
| **中** | 特定セクター・銘柄に大きく影響し得るイベント（個別企業決算、セクター規制変更） |
| **低** | 参考情報として把握しておくべきイベント（定例発表、影響が限定的な指標） |

### 注意事項

- **影響度「高」のイベント直前は新規ポジションの構築を控えることを推奨**する
- イベント通過後の方向性確認を待ってからエントリーする方がリスクが低い旨を付記する
- 地政学リスクは発生確率が低くても影響が大きい場合は「高」と判定する

## Step 4: 有望セクター特定

マクロ環境とイベントカレンダーから**3〜5セクター**を選定し、理由を明記する。

```
| 優先順位 | セクター | 注目理由 |
|----------|----------|----------|
```

## Step 5: 銘柄スクリーニング

### スクリーニング条件

| 条件 | 日本株基準 | 米国株基準 |
|------|-----------|-----------|
| 時価総額 | 1,000億円以上 | $10B以上 |
| 流動性 | 日次平均出来高が十分 | 同左 |
| 財務健全性 | 自己資本比率40%以上 or ネットキャッシュ+ | 同左 |
| 収益性 | ROE 8%以上、営業利益率が業界平均以上 | 同左 |
| バリュエーション | PER・PBRが業界平均比で割高すぎない | 同左 |
| 配当 | 利回り2%以上 or 連続増配実績（優先） | 同左 |
| モメンタム | 下降局面でないこと | 同左 |

**日本株・米国株それぞれ3〜5銘柄**を選定する。

## Step 6: 個別銘柄の詳細分析

各銘柄について以下を提示する:

### 基本情報
- 銘柄コード / ティッカー（日本株は `XXXX.T` 形式）
- 企業名、セクター、時価総額、直近株価

### 必須リスク指標（過去5年以上のデータ）

| 指標 | 値 |
|------|-----|
| CAGR（年平均成長率） | |
| シャープレシオ | |
| ソルティノレシオ | |
| 最大ドローダウン | |
| ボラティリティ（年率標準偏差） | |
| 勝率（月次ベース） | |
| 主要指数との相関係数 | |

### ファンダメンタルズ
- PER / PBR / EV/EBITDA
- ROE / ROA / 営業利益率
- 自己資本比率 / D/Eレシオ
- 配当利回り / 配当性向 / 増配年数
- 直近決算サマリー（売上・利益の前年比）

### 投資テーマ・カタリスト
- なぜ今注目に値するのか（具体的な材料）
- 成長ドライバー

### リスク要因
- 銘柄特有のリスク（競争環境、規制、為替、集中リスク）
- 想定下落シナリオとその影響度

## Step 7: モデルポートフォリオ提案

**コア・サテライト戦略**で提案する:

| 区分 | 配分目安 | 内容 |
|------|---------|------|
| コア（50〜70%） | インデックスETF | 全世界株式、S&P500、TOPIX等 |
| サテライト日本株（15〜30%） | 個別株（各5〜10%） | 選定した日本株 |
| サテライト米国株（5〜15%） | 個別株（各3〜5%） | 選定した米国株 |
| 現金・ゴールド（5〜10%） | リスクヘッジ | 急落時の買い増し余力 |

### ポートフォリオのルール
- 1銘柄あたり**最大10%**
- **損切りライン**: 各銘柄 -7〜10%
- **リバランス**: 四半期ごと
- **NISA活用**: つみたて投資枠 → インデックス、成長投資枠 → 個別株

## Step 8: 総合コメント

日本語200〜300字で以下を含める:
- 市場環境の総括
- 推奨戦略の要点
- 主要なリスク要因
- エントリータイミングのアドバイス（一括投入を避け、ドルコスト平均法を推奨）

## 出力フォーマット

1. **免責事項**（冒頭に必ず記載）
2. **マクロ環境サマリー**（表形式）
3. **直近の注意イベント**（日付・影響度付きの表形式）
4. **注目セクター一覧**（表形式）
5. **推奨銘柄一覧**（日本株・米国株別、表形式）
6. **各銘柄の詳細分析**（Step 6の全項目）
7. **モデルポートフォリオ**（配分比率の表）
8. **総合コメント**（200〜300字）

### 表示ルール

- リスクに関する数値は **赤系の色** で強調する
- リターンに関する数値は **緑系の色** で表示する
- チャートには適切なタイトル・軸ラベル・凡例を付ける
- 日本株の銘柄コードは `XXXX.T` 形式で記載する

## 注意事項

- **断定的な推奨はしない**
- データソースの信頼性と制約を明示する
- 海外銘柄を含む場合は**為替リスク**に必ず言及する
- 日本の税制（譲渡益税20.315%、配当課税等）を考慮に含める
- レバレッジ商品や短期トレード向け銘柄は除外する
- 個人の資産額・口座情報等の機密情報は出力に含めない

## クイックモード

ユーザが「とりあえず買うべき銘柄を教えて」等の簡易リクエストの場合:

1. マクロ環境は1〜2行で簡潔にまとめる
2. **最優先の1銘柄**（最も低リスクで汎用的なもの）を提示
3. **バランス型3銘柄**（セクター分散・地域分散を考慮）を提示
4. 買い方のポイント（分割買い、損切りライン、NISA活用）を箇条書き
5. **直近の注意イベント**を必ず表形式で提示する（以下の簡易フォーマット）:

```
| 日付 | イベント | 影響度 | 想定される影響 |
|------|---------|--------|---------------|
```

- クイックモードでも**今後2〜4週間の重要イベントを最低3〜5件**は収集・提示する
- 影響度「高」のイベントがある場合は、そのイベント前後のエントリータイミングについてコメントを付記する
- Web検索クエリ例: `株式市場 注目イベント 経済指標 今週 来週` / `FOMC 日銀会合 決算発表 スケジュール`

フル分析は「詳しく分析して」と言われた場合にStep 1〜8を実行する。
